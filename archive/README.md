# Archive

Local mirrors of old blogs in case the hosted versions ever disappear. Captured 2026-06-01 with `wget --mirror`.

| Blog | Period | Source |
|------|--------|--------|
| [Medium](https://medium.com/@matt.d.everard) | Mar 2017 – Dec 2018 | Medium (RSS) |
| [EntrepreNPM](https://entreprenpm.wordpress.com/) | Jun – Aug 2006 | WordPress.com |
| [Mountains and Ocean](https://mountainsandocean.blogspot.com/) | Mar – Jun 2006 | Blogger |
| [Thirsty Scholar](https://thirstyscholar.blogspot.com/) | Nov 2005 – Jan 2007 | Blogger |

Each folder contains the rendered HTML with assets and inline post pages. Open any `index.html` in a browser to read. Medium was scraped via RSS because Cloudflare blocks direct fetches; the original `feed.xml` is included alongside the per-post HTML files.

Not currently published to matteverard.org. The Writing section on the live site links to the original hosted URLs.
